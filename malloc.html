`malloc` btw is just a void pointer: void *malloc(size_t size); and so it returns a pointer to the allocated memory block. Thats it. offsets (specifically `offsetof()`) helps in determining the correct size to allocate when using memory allocation (*malloc, calloc, etc..*) for a structure: struct Point { int x; int y; }; struct Point *myPoint = malloc(sizeof(struct Point)); you then might use offsetof like: size_t structSize = sizeof(struct Point); offsetof(struct Point, y); youll be able to use myPoint and yOffset to access y member
